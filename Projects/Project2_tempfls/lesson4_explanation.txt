УРОК 4 — пояснение по коду

WorldBuilder
Создаёт весь игровой мир. Внутри метода CreateWorld создаются локации, настраиваются связи между ними, добавляются объекты и события. Также устанавливается стартовая локация игрока.

CreateWorld(GameState state)
Создаёт и настраивает:

* локации (Hall, Storage, DarkCorridor, GeneratorRoom, Exit)
* переходы между ними
* объекты (сундук, дверь, ловушка, генератор)
* события (урон в темноте, открытие выхода)
  Возвращает стартовую локацию.

AddExitEffect
Эффект, который добавляет новый переход между локациями. Используется для открытия новых путей (например, после открытия двери или включения генератора).

Apply()
Добавляет новую локацию в список выходов текущей локации.

ChangeLocationEffect
Эффект, который перемещает игрока в другую локацию.

Apply()
Меняет CurrentLocation в GameState.

OnEnterLocationEvent
Событие, которое срабатывает при входе в локацию.

OnTurnEvent
Событие, которое может срабатывать каждый ход (подготовка под игровой цикл).

OneTimeEvent
Событие, которое срабатывает только один раз.

Execute()
Проверяет условие и применяет эффекты, после чего больше не срабатывает.

ЛОКАЦИИ (Location)
Хранят:

* название
* описание
* объекты
* события
* выходы

Objects
Список объектов, с которыми можно взаимодействовать.

Exits
Список доступных переходов в другие локации.

Events
Список событий, которые проверяются автоматически.

TriggerEvents(GameState state)
Запускает все события локации.

ОБЪЕКТЫ

Chest
Контейнер с предметами.

Interact()
Выдаёт предметы (эффекты) один раз.

Door
Дверь с условием.

Interact()
Проверяет условие (например, наличие ключа). Если выполнено — применяет эффекты (открывает проход).

NPC
Объект (персонаж или терминал).

Interact()
Проверяет условие и выполняет эффекты (например, включает генератор).

Trap
Ловушка.

Interact()
Наносит урон один раз.

СОБЫТИЯ В МИРЕ

DarkCorridor событие
При входе без Torch наносит урон.

GeneratorRoom событие
После включения генератора открывает путь к выходу.

GameState (используется)
Хранит состояние игры:

* здоровье
* инвентарь
* флаги
* текущую локацию

Используется всеми условиями, эффектами и событиями для проверки и изменения состояния.
